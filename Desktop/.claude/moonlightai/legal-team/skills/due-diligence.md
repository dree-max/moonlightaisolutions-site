# Due Diligence Skill

## Purpose
Execute comprehensive transaction due diligence with systematic checklists, issue identification, and organized analysis for M&A, investments, and other transactions.

## When to Use
- M&A transactions (asset purchases, stock purchases)
- Venture capital investments
- Debt financing due diligence
- Partner/vendor vetting
- Real estate transactions

## Due Diligence Categories

### 1. Corporate & Organizational
- [ ] Formation documents (articles, bylaws, operating agreements)
- [ ] Good standing certificates
- [ ] Ownership structure and equity
- [ ] Board composition and minutes
- [ ] Shareholder/member agreements
- [ ] Authority to execute transaction

### 2. Financial
- [ ] Audited financials (last 3 years)
- [ ] Interim financials
- [ ] Revenue recognition policies
- [ ] Debt obligations and covenants
- [ ] Accounts payable/receivable
- [ ] Inventory and assets
- [ ] Tax returns (last 3 years)

### 3. Legal & Compliance
- [ ] Pending litigation
- [ ] Historical litigation outcomes
- [ ] Regulatory compliance
- [ ] Permits and licenses
- [ ] Material contracts
- [ ] Intellectual property
- [ ] Employment matters

### 4. Operational
- [ ] Material customers (concentration)
- [ ] Key suppliers
- [ ] Real property leases
- [ ] Equipment and assets
- [ ] IT systems and data
- [ ] Insurance coverage

### 5. Employment & Benefits
- [ ] Employee roster and terms
- [ ] Employment agreements
- [ ] Benefits plans
- [ ] Independent contractor arrangements
- [ ] OSHA/compliance

### 6. IP & Technology
- [ ] Patents and trademarks
- [ ] Copyrights
- [ ] Trade secrets
- [ ] Software licenses
- [ ] Data privacy compliance
- [ ] Open source usage

### 7. Environmental (if applicable)
- [ ] Environmental permits
- [ ] Historical compliance
- [ ] Environmental liabilities

## Analysis Framework

### Issue Categorization
- **SHOWSTOPPER**: Issues that must be resolved or buyer must walk
- **MATERIAL**: Significant issues requiring negotiation or escrow
- **MODERATE**: Issues to be aware of and potentially address
- **MINOR**: Informational items

### Risk Assessment
For each issue identified:
1. What is the issue?
2. What's the legal/financial exposure?
3. How likely is it to occur?
4. Can it be mitigated? How?
5. What is the mitigation cost?

### Summary Format
```
## Due Diligence Summary - [Target Company]

### Overall Assessment
[2-3 sentence summary]

### Showstopper Issues
- [Issue 1]
- [Issue 2]

### Material Issues
- [Issue 1 with analysis]

### Moderate Issues
- [Brief list]

### Recommendations
1. [Recommendation 1]
2. [Recommendation 2]

### Items for Further Investigation
- [Open items]
```

### Checklist Tracking
Maintain a living checklist:
- Items reviewed
- Documents obtained
- Issues identified
- Status (cleared, flagged, pending)

## Quality Standards
- Document every assumption
- Flag jurisdiction-specific issues
- Note information gaps
- Cross-reference issues between categories
- Quantify financial exposure where possible

## Output
- Executive summary
- Detailed issue analysis
- Prioritized recommendations
- Clean checklist with status
- Document request list (if ongoing)
